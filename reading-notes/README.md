# 📝 读书笔记

本目录用于存放通过深度阅读系统生成的读书笔记。

---

## 📂 目录结构

```
reading-notes/
├── README.md              # 本文件
├── templates/            # 笔记模板
│   ├── 理论类书籍模板.md
│   ├── 案例类书籍模板.md
│   └── 传记类书籍模板.md
│
├── in-progress/          # 正在阅读的笔记
│   └── .gitkeep
│
└── completed/            # 已完成的读书笔记
    └── .gitkeep
```

---

## 📖 使用流程

### Step 1: 选择模板

根据书籍类型，选择合适的模板：

- **理论类书籍**（如《从0到1》《原则》）  
  → 使用 `templates/理论类书籍模板.md`

- **案例类书籍**（如《巴菲特致股东信》《Jeff Bezos致股东信》）  
  → 使用 `templates/案例类书籍模板.md`

- **传记类书籍**（如《硅谷钢铁侠》《乔布斯传》）  
  → 使用 `templates/传记类书籍模板.md`

### Step 2: 在Claude Projects中阅读

1. 打开Claude Projects："思考，推理，读书"
2. 输入："我想读《书名》"
3. 按照AI导师的引导，完成深度对话
4. AI会自动生成笔记内容

### Step 3: 保存笔记

1. 将AI生成的内容复制到模板中
2. 保存到 `in-progress/` 文件夹
3. 命名格式：`书名-作者.md`（如：`从0到1-彼得蒂尔.md`）

### Step 4: 完成后归档

阅读完成后，将笔记移动到 `completed/` 文件夹。

## 📋 笔记模板说明

### 理论类书籍模板

适用于讲解理论、方法论、思维模型的书籍。

**核心结构**：
- 书籍档案
- 核心理念
- 仓库理论映射
- 实战练习
- 智慧卡片
- 行动计划

### 案例类书籍模板

适用于商业案例分析、公司传记、行业研究的书籍。

**核心结构**：
- 书籍档案
- 核心案例
- 决策分析
- 仓库理论验证
- 对标学习
- 行动启示

### 传记类书籍模板

适用于人物传记、创业历程、领导力故事的书籍。

**核心结构**：
- 书籍档案
- 人物画像
- 关键转折点
- 思维模式提炼
- 仓库方法论对比
- 可复用法则

## 🔗 与知识库的关联

每篇笔记都会自动关联 [三湘问道知识库](https://github.com/MRYGP/sanxiangwendao)：

### 关联方式

**书中理论** ← → **仓库文档**

示例：
```
书籍：《从0到1》
核心理论：垄断思维、第一性原理

关联仓库：
- DOC-S037：从0到1（核心解读）
- DOC-S001：创新三元法
- DOC-S036：规则制定者

关联类型：互补/对比/验证
```

## 📊 笔记质量标准

### 优秀笔记的特征

✅ **深度理解**：
- 不只是摘抄，而是深度理解
- 能用自己的话解释核心概念
- 能举出新的例子验证理论

✅ **仓库融合**：
- 明确标注与仓库理论的关联
- 说明两者的关系（互补/对比/验证）
- 给出跨理论的深度洞察

✅ **实战导向**：
- 完成深度练习题
- 制定具体的行动计划
- 记录实践过程和反思

✅ **结构清晰**：
- 遵循模板结构
- 逻辑连贯
- 易于回顾

## 🎯 使用建议

### 建议1：边读边记

不要等读完再写笔记，而是：
- 每完成一个模块，立即记录
- 记录你的思考过程
- 记录你的疑问和灵感

### 建议2：定期回顾

- 每周回顾本周的笔记
- 每月回顾本月的所有笔记
- 寻找跨书籍的联系

### 建议3：实践验证

- 将笔记中的理论应用到实际工作
- 记录应用效果
- 更新笔记中的"行动计划"部分
