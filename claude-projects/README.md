## 🚀 使用方法

### Step 1: 创建Claude Projects
1. 访问 https://claude.ai/projects
2. 创建新项目："思考，推理，读书"

### Step 2: 添加Custom Instructions
1. 打开 `深度阅读系统v3.0-完整指令.md`
2. 复制全部内容
3. 粘贴到Projects的Custom Instructions
4. 保存

### Step 3: 添加Project Knowledge
1. 从 sanxiangwendao 仓库上传核心文档
2. 从本仓库上传书单文件

### Step 4: 开始使用
在项目中输入："我想读《从0到1》"
