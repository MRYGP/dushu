# 🎯 实战练习

本目录用于记录深度思考练习的题目和答案。

---

## 📂 什么是实战练习

实战练习是深度阅读系统的核心环节，通过5-10分钟的深度思考题，将书中理论内化为可用的认知能力。

### 练习类型

**类型1：案例应用**
- 给出一个真实复杂场景
- 用书中理论 + 仓库理论综合分析
- 至少500字的深度分析

**类型2：思维模拟**
- AI扮演书中大师 + 仓库理论专家
- 你提出问题，AI从多个视角回答
- 对比不同理论的视角差异

**类型3：对比分析**
- 两个相似场景的辨析
- 结合两个理论框架分析
- 至少400字的对比分析

## 💡 使用建议

### 建议1：认真思考
- 不要急于提交答案
- 至少花5-10分钟思考
- 写下完整的分析过程

### 建议2：对比学习
- 提交答案后，认真看标准答案
- 找出你的盲区
- 理解WHY（为什么这个答案更好）

### 建议3：定期回顾
- 每周回顾本周的练习
- 查看得分趋势
- 发现自己的进步

## 📊 评分标准

### 4个维度（总分100分）

**维度1：理论应用（40分）**
- 是否正确理解书中理论
- 是否准确应用理论
- 是否深度分析

**维度2：仓库融合（30分）**
- 是否关联仓库理论
- 是否说明两者关系
- 是否产生跨理论洞察

**维度3：分析深度（20分）**
- 是否多层次分析
- 是否考虑边界条件
- 是否给出具体建议

**维度4：可执行性（10分）**
- 建议是否可执行
- 步骤是否清晰
- 是否考虑约束
