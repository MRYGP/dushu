# 🎯 实战练习

本目录用于存放深度思考练习题及答案。

---

## 📂 什么是实战练习

实战练习是深度阅读系统的核心环节，通过**5-10分钟的深度思考题**，将书籍理论真正内化为认知能力。

### 练习特点
- ✅ **深度思考**：不是简单应用，而是需要综合分析
- ✅ **理论融合**：结合书中理论 + 仓库理论
- ✅ **实战导向**：模拟真实场景，可直接应用
- ✅ **标准答案**：AI给出详细解析和评分

---

## 📋 练习类型

### 类型1：案例应用（深度分析）

**特点**：
- 给出复杂的商业场景
- 要求用理论框架分析
- 给出明确的决策建议

**示例**：
```
假设你是2010年的贝索斯，面对：
- Kindle业务连续3年亏损
- 董事会要求砍掉Kindle
- 但你认为Kindle是未来

请用以下框架分析：
1. 巴菲特的"护城河"理论
2. 仓库理论《时间的朋友》
3. 仓库方法《三大标杆方法论》

要求：至少500字，结合两个理论框架
```

---

### 类型2：思维模拟（角色扮演）

**特点**：
- AI扮演书中大师 + 仓库理论专家
- 用户提问，AI用综合思维回答
- 对比不同思维模式的差异

**示例**：
```
现在，我将同时扮演：
- 查理·芒格（多元思维模型大师）
- 松阳（仓库理论创建者）

你可以向我们提问任何商业/投资/决策问题。
我们会从两个角度给你答案，并指出异同。
```

---

### 类型3：对比分析（深度辨析）

**特点**：
- 给出两个相似但本质不同的场景
- 要求用理论框架辨析
- 给出明确的选择和理由

**示例**：
```
以下两家公司，哪个更符合巴菲特的"护城河"标准？

公司A：市占率60%，技术门槛不高，利润率15%
公司B：市占率20%，专利技术10年独占，利润率30%

请分析：
1. 用巴菲特的"护城河"理论判断
2. 用仓库理论《规则制定者》对比
3. 如果你是投资者，选哪个？
```

## 📊 练习流程

### Step 1: 完成练习
在Claude Projects中，阅读完理论后，AI会给出练习题。

### Step 2: 深度思考
用5-10分钟，认真思考并写下答案（至少400-500字）。

### Step 3: 提交答案
输入`【提交答案】`，AI会给出标准答案和详细解析。

### Step 4: 查看评分
AI会从4个维度评分（总分100分）：
- 理论应用准确性（40分）
- 仓库理论融合度（30分）
- 分析深度（20分）
- 决策可执行性（10分）

### Step 5: 改进学习
根据AI的反馈，找出差距，下次改进。

## 💡 使用建议

### 建议1：认真对待每次练习
- 不要敷衍，每次至少写400-500字
- 深度思考，而非表面应用
- 结合书中理论 + 仓库理论

### 建议2：保存所有练习记录
- 用于回顾和复盘
- 看到自己的进步
- 发现常见盲区

### 建议3：定期回顾练习
- 每月回顾本月的所有练习
- 对比不同练习的答案质量
- 总结常见问题和改进方向

### 建议4：实践中验证
- 将练习中的思考应用到实际工作
- 记录应用效果
- 持续迭代认知

## 📊 练习质量标准

### 优秀答案的特征（85分以上）

✅ **理论应用准确**：
- 准确理解书中理论
- 正确引用仓库理论
- 两者有效融合

✅ **分析深度足够**：
- 不只是表面描述
- 有深层次的洞察
- 能举出新的例子

✅ **决策可执行**：
- 给出明确的建议
- 建议具体可操作
- 考虑了实际约束

✅ **结构清晰**：
- 逻辑连贯
- 分点论述
- 有总结
